# Memow

Application PWA React + TypeScript avec Vite et Tailwind CSS.

## Installation

```bash
npm install
```

## Exécution en développement

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Fichiers ajoutés

- `package.json`
- `tsconfig.json`
- `tsconfig.node.json`
- `vite.config.ts`
- `index.html`
- `src/main.tsx`
- `postcss.config.js`
- `tailwind.config.js`
- `public/manifest.webmanifest`
- `public/favicon.svg`
- `public/icons/icon.svg`
- `.gitignore`
