# Memow 🧠

**Entraîne ta mémoire, pas juste ta lecture**

Memow est une Progressive Web App (PWA) qui aide à mémoriser du vocabulaire grâce au principe de **rappel actif** (active recall) et à la **répétition espacée**.

## Fonctionnalités

- 📱 **Offline-first** - Fonctionne sans connexion internet
- 🧠 **Rappel actif** - Réfléchis avant de voir la réponse
- 🔁 **Répétition intelligente** - Les mots difficiles reviennent plus souvent
- 📊 **Statistiques** - Suis ta progression
- 🇬🇧🇫🇷 **Multi-langue** - Anglais et Français
